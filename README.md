# FIRA AI-Agent 2017

서울대학교 빅데이터 연구소(BDI) [인공지능 에이전트 과정](http://bdi.snu.ac.kr/academy/?page_id=52) 수강 기록(2017.06-2018.02)

    + 음성 인식 - 성원용 교수님

    + 기업 혹은 대학원 연구실에서 프로젝트 진행
